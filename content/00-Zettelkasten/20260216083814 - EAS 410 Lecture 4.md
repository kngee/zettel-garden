- DMA (Direct memory Access) is a powerful operation that a CPU and I/O device can use to directly transfer data between one another. There is no longer a need to tie up the processor and this is a major speed up. It is no longer the processor's responsibility to facilitate the exchange.

- Modern systems are moving away from busses because of the amount of data we require to transfer between points. Point to point will always be faster than a shared general bus. 
- A bus is any communication pathway between two or more devices. Requires that only one device may transmit at a time. System buses are internal and connect major computer components. ![[Pasted image 20260216084544.png|center]]
- Data busses are the most high bandwidth. We characterise busses by their width. Each bit requires 2 lines, the number of lines is referred to as the width.
- The address bus holds the target instruction addresses on here to signal that someone wants to request the data. The width determines the maximum memory capacity of the system.
- The control bus is the fed, it controls access and transmits the command and timing information. We need timing to indicate the validatity ![[Pasted image 20260216085230.png|center]]
- There are physical contraints to a shared bus and this motivated the move to point-to-point connections. Even inside a chip we cannot have busses, and have opted for point to point connections. ![[Pasted image 20260216085535.png|center]]
