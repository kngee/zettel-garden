- Model-Based Systems Engineering can be used to model a complex system from conception to the end of its life.
- A system is rigourous if it has a common language, easy abstraction of information and have automated checks for completeness and correctness.
	- SysML offers an open standard for systems modelling.
	- Abstraction allows us to select the aspects that are relevant to answer the questions for the current stage of development.
	- Automation ensures that designs are validated for quality and consistency
- Gaphor is a lightweight option if our main concern is adhering to SysML standards.
