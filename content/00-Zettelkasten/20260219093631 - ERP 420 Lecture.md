> Introduction to sensor fusion and probability theory
---
- What is fusion, and its three levels.
- An example of sensor fusion begins with sensing, then perceiving, planning and then acting.
- Sensor fusion encompasses the sense and perceive functions.
- Statistical Sensor Fusion is based in probability theory
- We need probability theory to handle random, noisy or uncertain phenomena.
- Probability Space 
	- $\Omega$, considers the sample space which is the set of all possible outcomes.
	- $F$, a set of events where each event is a set containing zero or more outcomes.
		- An event should be a subset of the sample space
	- $P$, probability measure, a function from events to probabilities. There will always be an outcome to the experiment.
	- There is an example of the probability space in the slides for the case of flipping a coin.
- A random variable X is the a measureable function from the sample space towards the state space. This variable can be discrete or continuous. We have specific outcomes and we can now map them to values.
![[Pasted image 20260219101112.png|center]]
- Random variables are super useful and can be defined to be useful to the researcher.
- Probability Distribution lists all the possibile outcomes of a random variable along with their corresponding probability values
- Some elementary rules we need to revise are the *Product Rule* and the *Sum Rule*
- Joint, Marginal and Conditional probabilities are the most important probabilities for considering **more than 1 random variable**
- Joint probability $p(X,Y)$ when we consider the values of X and Y at the same time
- Marginal probability considers individual variables.
- Using the joint probabilities, we can derive the marginal probability...very easily
- If our variables were continuous, the summations would be integrals.
- 1 variables value given the specific value of a another variable which is the conditional probabilty.
- If you have already defined the confitional probability and you have marginal probability then you can get the joint probability.
- **Bayes Rule** is everything. We derive it by combining the foundational rules we can derive it.
![[Pasted image 20260219101652.png|center]]
- Probability densities help us show the probability measures for a really large number of outcomes, say for continuous data, there are continuous and discrete forms of the probability density.
- Understand that when working with probability densities, we say *what is the probability that it is within a certain range*
- **Values of a PDF are not probabilities**
- *In the case of a discrete PDF, you can read the probability, BUT YOU CANNOT DO THAT.*
- Probability densitives have their own set of functions for sum and product rules
- **Expectation** is one of the most important characteristics in probabilty
- Help us have an understanding of the PDF, it gives us the average value of $f(x)$ under a probability distribution $p(x)$
- It has corresponding functions for the discrete distribution and the continuous distribution.
- **Variance** is the second most important characteristic. Variance can be written in terms of **Expectation**. Remember that it measures variability around the mean.
