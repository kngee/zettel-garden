In this module, we aim to learn digital signal processing algorithms and procedures that will help us analyse, design and implement for a real world applications. We will learn about and implement DFT, FFT, Cyclic Convolution and Correlation, Overlap-and-Add, Overlap-and-Store. We will also learn about implement digital filters. Additionally, we learn about system simulation skills for DSP.
