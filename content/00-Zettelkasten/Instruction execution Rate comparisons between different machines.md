This question was found Tutorial 1 as question 2. The question compares two different machines each with their own instruction set and both have a clock rate of $200~\text{Mhz}$. In the first part of the question I need to determine the *CPI*, *MIPS Rate* and the *Execution Time* for both machines. After that we need to be able to calculate the *MIPS* for each computer. The question also wants us to using different means to compare the computers based on these means.
