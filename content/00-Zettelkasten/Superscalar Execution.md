Superscalar prediction is a technique to speed up the execution of tasks by a microprocessor. This technique is similar to pipelining in that allows the processor to issue more than 1 instruction in each clock cycle which you can think of allows you to have **multiple** parallel pipelines.

If your troops are instructions then the barracks are an implementation of superscalar execution.
