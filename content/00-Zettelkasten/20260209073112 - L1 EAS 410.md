- Computer Architecture refers to the attributes of the system that are visible to the programmer - these decisions have a direct impact on the logical execution of the program. **We also call this Instruction Set Architecture**.
- ISA should make you think about the instructions that are visible to the programmer. What can you do vs How it should be achieved.
- Computer Organisation refers to the units and interconnections that exist that make up the architecture. These hardware details are transparent to the programmer. 
	- Ie. Control signals, Memory technology etc etc. 
- Architecture doesn't change as fast as organisation.

> Structure & Function

- Hierarchical approach is really important and allows the designer to focus on one stage at a time.
- The structure refers to how components relate to one another
- The funcion is the operation of individual components as part of the structure. 
- There are 4 basic functions of a computer: 
	- Data processing - one of the most fundamental
	- Data storage
	- Data movement
	- Control
- The 3 latter functions pretty much just serve data processing.
- There are 4 main structural components of a computer
	- CPU
		- Its main components are the: Control Unit, ALU, Registers, CPU interconnections.
	- Main Memory: Note that DDR (Double Data Rate) because data is accessed twice per oscillation.
	- I/O
	- System Interconnection

> Multicore Structure

- Each core can do single CPU tasks. 
- Multiple cores on a single piece of silicon is referred to as a processor.
- The process is what eecutes the instructions.
- A core can have multiple threads.

> Cache Memory

- There are multiple layers of memory between the processor and the main memory
- Everything is motivated by lowering latency
- Places memory likely to be used in the future in this so that it can be quickly retrieved.
- Multiple levels of cache can be found to increase performance.

> Yield as a performance issue

- Silicon provides faster interconnects
- However, the larger the block of silicon, the higher the chance of holes that reduce throughput

The whole computer system tries to make sure that the ALU can operate at maximum efficiency.

> 1.4 GATES, MEMORY CELLS, CHIPS, AND MULTICHIP MODULES

- Gate is a device that implements a simple boolean function
- A memory cell is a device that can store one bit of data.
	- A memory cell has 4 basic functions much like a larger structure which are: Data storage, Data Processing, Data Movement and Contorl
- It is by interconnecting many of these units that we are able to build a computer.
-  ![[Pasted image 20260209084504.png | center ]]
- Transistors work because they allow conditional voltage paths.
- Many transistors can be produced at the same time on a single wafer of silicon. Equally important, these transistors can be connected with a process of metallization to form circuits.
- Over time the transistors packed into a single silicon die grew exponentially.
- Moore's law: Moore observed that the number of transistors that could be put on a single chip was doubling every year, and correctly predicted that this pace would continue into the near future. To the surprise of many, including Moore, the pace continued year after year and decade after decade. The pace slowed to a doubling every 18 months in the 1970s, but has sustained that rate ever since.
	- Cost of computer logic and memory circuit has fallen dramatically
	- Because logic and memory elements are placed closer together on more densely packed chips, the electrical path length is shortened, increasing operating speed.
	- The computer became smaller making it more accessible
	- There is a reduction in power requirements
	- The interconnections on the integrated circuit are much more reliable than solder connections. With more circuitry on each chip, there are fewer interchip connections.
- Motivation for multichip modules is that we want to decrease the spacing between ICs in a system which increases performance and reduces the noise.
- MCMs are classified by the chosen substrate.
- MCMs have a set of 5 components in its architecture.
